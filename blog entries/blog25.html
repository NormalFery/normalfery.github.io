PrideGUI 3.0.0 (& Legacy 2.0.0)

So, after a few issues regarding mod compat, and a lot of struggling, I'll probably regret saying this later but PrideGUI 3.0.0 should be ready for public release sometime soon! Along with Legacy 2.0.0, which is the version with a lot of the compat

Before the list, if there's an "(M)" next to the mod name, it's because the mod is in both main (1.20.2+), and Legacy (1.20.1), note that if it doesn't have the symbol, it just isn't in main

In order, the mod compat added has been :

  • No Chat Reports (M)
  • Fabric API (the switching buttons for the creative menu)
  • Roughly Enough Items and Eek! My Items! (EMI)
  • Supplementaries
  • Amethyst Imbuement
  • Axiom (M)
  • Sidekick (M)
  • Litematica (M)
  • Inventory Profiles Next (M)
  • Create 0.5.x (Until 6.0.0 releases in Fabric)

I want to kinda thank ThinkSeal for being the one to open all the issues and do all the testing xd

Aside from that, unless any more issues pop up before release, hopefully it should be released soon!
