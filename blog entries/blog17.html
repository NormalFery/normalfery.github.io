More MRCL! A lot of MRCL...

Yup, I've been playing wayyyy more MRCL, how much? Well... A lot! Last I was here, I think I said I got to beta test some features alone, but now... I got to beta test features with other veterans and patreon members! It was chaotic lmao. We did find a bunch of bugs, and some stuff that should be improved (I managed to find a logic flaw!) and the Elevated Functions update released! Now it's public for everyone to play! It was great seeing how the devs made progress and the fact that they established a robust base to expand is really nice for the future of the server! Now the next updates will be quite far away, since first there's a Pre-update checklist. But I'm sure that the Digital Sea will see it's glory sooner than later! If you guys wanna see any of the updates, you should check on the MRCL Wiki! Why? Well... Because I help translate it to Spanish :3 (I've also written the Customization and Interface Programs pages!)
