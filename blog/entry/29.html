Howdy Everyone! Fery back here, for big announcements about the website! Because "FerySite 2", as I'm calling it, is almost entirely feature complete! You might know some of these things already if you follow my bluesky, but even then, here's more of my insight! Ready? Yes? Then lets get started!

Navbar changes

Yup! Believe it or not, the navbar has changed! Alongside new buttons, the way the navbar is handled in code is different too!

If you want to skip the technical part, click me for the TL;DR! Otherwise, go on!

So pretty much, what I've done (with some help from, NOT AI, but google), specifically taking code from this "HowTo"from W3 Schools, I made it so instead of each page having a single navbar that HAS to be changed in EVERY page to avoid outdated pages, I only have to change 3 html files! One for each "level" Blogs (and Memories) use the deepest level, while the index uses the surface one!

TL;DR: now each page just calls an html file, and a new js script makes it appear.

I may not have written the code itself, but troubleshooting it was painful enough. I will probably reuse the script but for other things in the future! Maybe make the "head" part of the page also different? (Except title and color CSS)

Talking about CSS...

CSS "Splits"

After a LONG time of keeping most of the CSS the same, big as fuck files, I finally decided to use variables for the CSS instead! Now you are loading "main.css", which gives the structure, and "theme.css", which gives the page the color it needs!

This way, if I need to fix a div, or a grid, I don't need to do it ALL 5 TIMES for themes, and theme implementations are now easier!

If you wanna add your own theme to the site, feel free to do a PR! Just take a look at the theme files and change the colors around ^^

Grids? Grids. Grids!

If you've been to the music page recently, you might've seen the newer style!

This is all thanks to CSS Grids and a lot of pain! And it was worth it! Now we have 3 songs per row, since that's what I usually drop in each blog entry!

I will try to update it every so often without depending on blog entries though, so make sure to check them!

Page Merging, Changes, and more

After a lot of thinking, I decided to do a move to make the site have... "less" pages. How? By merging them!

This was done with,

  • The Blog & Memories Pages (made into a single blog page)
  • The "Who", "Where", and "Projects" page, made into a single "About Me" page

These pages also use CSS Grid! The Blog and Memories Pages uses it to split the lists into the Blog and Memory Sections, while the "About Me" (internally called "who.html" by the way), uses it for 3 "bullet points" that answer the questions easily.
